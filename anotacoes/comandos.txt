trazendo so arquivos do breeze para dentro da aplicação 

   composer require laravel/breeze --dev

instalando o breeze 

   php artisan breeze:install

criando a migrations dos posts 
   
   php artisan make:migration create_posts_table

criando as tabelas executando as migrations

   php artisan migrate

criando um seeder para os usuarios 
  
  php artisan make:seeder UsersTableSeeder 

criando seeder para os posts 

  php artisan make:seeder PostsTableSeeder

rodando a seed pra cadastrar usuarios 

   php artisan db:seed --class=UsersTableSeeder

rodando a seed pra cadastrar posts 
  
  php artisan db:seed --class=UsersTableSeeder

criando o model posts 
  
   php artisan make:model Posts

criando o controller principal da aplicação 
   
   php artisan make:controller MainController

criando o componente Post 

   php artisan make:component PostComponent

criando a view pra cadastrar os posts 
   
   php artisan make:view create-post
